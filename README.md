# Features
With this mod, you can change the layout of the buttons, leaderboard and level info on the end screen.

Also on the end screen, you can make two custom backgrounds: one image for beating the level, and another image for when you get a new medal (New Medal background may not work perfectly if you are playing a level for the first time).

There is one new Player Portrait that only appears when you beat a level, you need to enable it, but it will not be affected by the gray filter on the end screen, and you can make it slide into the screen (Slide positions configurable) or just appear from the end position.

The Player Portrait and the other player image that shows during the end animation can be changed with NeonLite, but if you want to have a GIF on them specifically you can change that too. 

# How to Install

Take the ZIP file from the latest release and extract it. Everything in the Mods folder goes into the Mods folder from Melon Loader, everything from the UserLibs folder goes into the UserLibs folder from Melon Loader. (Magick.NET-License does not go into any folder)

The mod MIGHT work if you don't put any UserLibs (Please still add them so logs are smaller), but GIFs will not work and you may crash as it depends on Magick.NET.
