# NeonMenuCustomizer
Mod to customize menus in Neon White! You can add things such as custom backgrounds on the level end animation, a background if you get a medal, different player portraits on the end screen, and change the layout of panels in menus.
# IMPORTANT:
This mod has GIF support, but it will not work without Magick.NET. Inside of the release are libraries for Magic.NET, placed in the Magick.NET Libs folder. Take everything in Magick.NET Libs and put it into the UserLibs folder in the Neon White folder. NeonMenuCustomizer.dll inside the NeonMenuCustomzier folder goes into the Mods folder.
